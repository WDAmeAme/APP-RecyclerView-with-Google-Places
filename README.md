# Archivos

Los archivos contiene el Adaptador, Modelo de datos y Vista del RecyclerView, contiene la vista y la actividad principal: **Actividad/Prueba.java** usar la clave API en Strings valores.

## Librerías
- Volley 1.1.0
- Glide 4.7.1
- Places Library 1.0.0
- Service Maps 16.0.0

## Archivos JAVA

- Actividades:
	- Prueba (Principal).
- Adaptadores:
	- PruebaAdaptador.
- Modelos:
	- PruebaModelo.
- Peticiones:
	- Constantes.
- Utilidades:
