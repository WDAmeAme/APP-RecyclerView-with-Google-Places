# Archivos

Los archivos contiene el Adaptador, Modelo de datos y Vista del RecyclerView, contiene la vista y la actividad principal: **Actividad/Prueba.java** usar la clave API en Strings valores.

## Archivos JAVA

- Actividades:
	- Prueba (Principal).
- Adaptadores:
	- PruebaAdaptador.
- Modelos:
	- PruebaModelo.
- Peticiones:
	- Constantes.
- Utilidades:
	- MySingleton Volley.
